# OpenCV 3.x Examples all-in-one

This repo may be helpful for a beginners

## Requirements:

* OpenCV 3.1.0 - 3.2.0
* cmake >= 2.8

## Examples:

* CamShift
* CameraMotion
* CascadeFaceFind
* CompareTrackers
* Contourus
* HSVThreshold
* LoadModifySaveImage
* OpticalFlowLKMethod
* OpticalFlow
* SequentialImages
* ShowImage
* TemplateMatcher
* VideoCapture2Edges
* UseORB
* UseMultiTracker
